============================Настройка=============================
Для инициализации и настройки подключения к базе данных необходимо
указать соответствующие атрибуты строки подключения в файле 
"\CMSweb\CMSweb\Web.config"(секция "connectionStrings"):

1. Server - [IP-адрес сервера или dns-имя];
2. User Id - [Имя пользователя PostgreSQL(postgres)];
3. Password - [Пароль пользователя];
4. Port - [Порт(5432)]
5. Database - [Имя базы данных]

Базу данных создавать(вручную) не нужно.
==========================Web API================================
1. "/api/articles/GetLast/n" - получение списка последних n статей;
2. "/api/articles/id" - получение информации о статье по ее id

Примеры запросов:
localhost:64593/api/articles/GetLast/6 - список 6-ти последних статей;
localhost:64593/api/articles/3 - информация о статье с id=3
